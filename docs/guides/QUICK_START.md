## First Time Setup

### 1. Create Master Password
```
✓ 8+ characters minimum
✓ Mix of letters, numbers, symbols
✓ Something memorable but secure
✓ CANNOT be recovered if forgotten!
```

### 2. Add Your First Password
```
1. Click the + button
2. Fill in:
   - Title (e.g., "Gmail")
   - Username (e.g., "user@gmail.com")
   - Password (or click Generate)
3. Click Save
```

### 3. Create Your First Backup
```
1. Click Export (📤)
2. Save the .vault file
3. Store in safe place (iCloud, Drive, USB)
4. File is encrypted - safe to store in cloud!
```

## Backup & Restore

### Export (Backup)
```
1. Click 📤 Export
2. File saved: vault-backup-YYYY-MM-DD-HHmmss.vault
3. Store in multiple safe locations
```

### Import (Restore)
```
1. Click 📥 Import
2. Select .vault file
3. Enter master password
4. Review merge stats
5. Confirm import
```

### Auto-Backup
```
✓ Automatic after each change
✓ Keeps last 3 backups locally
✓ No action needed
✓ Recovery available if needed
```

## Security Tips

### ✅ DO
- Use strong master password
- Export backups regularly
- Store backups in multiple locations
- Test restore process occasionally
- Enable biometric unlock (mobile)

### ❌ DON'T
- Share master password
- Store master password in app
- Forget to backup
- Use weak passwords
- Store backups unencrypted (they already are!)

## Troubleshooting

### Can't unlock?
```
→ Check master password spelling
→ Try again carefully
→ If forgotten, restore from backup
```

### Import not working?
```
→ Verify .vault file format
→ Check master password
→ Ensure file not corrupted
```

### Backup reminder annoying?
```
→ Export a backup
→ Or click "Don't remind again"
```

### App won't open (macOS)?
```
→ Right-click → Open (first time)
→ This bypasses Gatekeeper
```
